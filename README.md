# 🤖 End-to-End AI Capstone

A Python-based AI assistant that combines **Face Recognition, Voice Commands, and Text Input** to perform real-world tasks like opening applications, browsing the web, and more.


## 🚀 Features

- 🔐 Face Authentication System
- 🎤 Voice Command Recognition
- ⌨️ Text Command Support
- 🌐 Open YouTube / Google
- 📝 Open Notepad
- ⏰ Display Current Time
- ❌ Handles Unknown Commands Gracefully


## 🛠️ Technologies Used

- Python
- OpenCV
- face_recognition
- SpeechRecognition
- pyttsx3
